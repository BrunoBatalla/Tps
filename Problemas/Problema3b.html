Programación: Es el proceso de crear un conjunto de instrucciones que le dicen a una computadora cómo realizar una tarea específica.

Algoritmo: Conjunto de pasos o reglas que se siguen para resolver un problema o realizar una acción.

Base de datos: Conjunto estructurado de datos que permite el almacenamiento, gestion y recuperacion de informacion. puede ser relacional o no y es escencial para aplicaciones que manejan grandes volumenes de datos.
